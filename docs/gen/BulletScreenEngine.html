Methods

addBulletScreen(bulletScreen)

添加弹幕到弹幕列表。由于弹幕在屏幕上出现过后，弹幕引擎将从列表中彻底删除此弹幕。所以，在改变播放进度时，可能需要先清空弹幕列表，然后重新加载此播放进度以后的弹幕。

Type Definitions

BulletScreen

BulletScreen 结构用于存放单条弹幕数据。

Type:
  • Object
Properties:
Name Type Argument Default Description
text String

弹幕文本
canDiscard Boolean <optional>
 true

是否允许丢弃：在弹幕过多时，程序将自动丢弃一些延迟过高的弹幕。此选项为 false 时本条弹幕无论如何都不会被丢弃，使用本选项的场景如本用户发送的弹幕。注意：不要将太多的弹幕的 canDiscard 设为 false， 否则会因超时的弹幕不会被丢弃而造成意外的问题。
startTime Integer <optional>
 options.clock()

弹幕进入时间：单位：毫秒，默认为时间基准（options.clock）当前时间。
type Integer <optional>
 0

弹幕类型：0：从右到左（普通）；1：从左到右（逆向）；2：顶部固定；3：底部固定。
style BulletScreenEngine~BulletScreenStyle

弹幕样式：一个 BulletScreenEngine~BulletScreenStyle 结构。设置此选项中的任何一个值，将覆盖对应的全局设置。
