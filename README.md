高性能弹幕引擎。同屏弹幕1000+

## 简介
一个高性能弹幕引擎，简单易用。支持CSS3、Canvas 2D、WebGL渲染方式。支持顶部、底部、逆向弹幕渲染。使用Canvas 2D渲染方式可流程渲染同屏1000条弹幕（可达1600条，与电脑配置有关）。
